# PingIt - Real-time Random Chat Web App

**PingIt** is a simple, real-time random chat web application built with a **Next.js frontend** and a **Node.js + Socket.IO backend**.

This app is **completely DB-less** and uses **temporary usernames only**, which are not stored on the server or in any database.
